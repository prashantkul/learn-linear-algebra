# Chapter 2 — Vectors and Vector Geometry

## Learning objectives

- Add vectors and scale them; interpret both operations geometrically.
- Compute and reason about the dot product (algebraic and geometric forms).
- Project one vector onto another and decompose a vector into parallel + perpendicular parts.
- Parametrize lines in ℝⁿ.
- Distinguish convex, affine, and linear combinations.

## Prerequisites

- Chapter 1.

## Estimated time

5–7 hours.
