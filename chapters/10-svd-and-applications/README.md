# Chapter 10 — Symmetric Matrices, Quadratic Forms, SVD, and Applications

## Learning objectives

- State and use the spectral theorem for symmetric matrices (orthogonal diagonalization).
- Classify quadratic forms (positive/negative definite, indefinite) and connect to conic sections.
- Compute the singular value decomposition; interpret singular values geometrically.
- Apply SVD to low-rank approximation, image compression, and PCA.
- (Bonus) Set up and solve a small system of linear differential equations using diagonalization.

## Prerequisites

- All previous chapters.

## Estimated time

10–12 hours.
