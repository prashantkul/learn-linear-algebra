# Chapter 9 — Eigenvalues, Eigenvectors, Diagonalization, Dynamical Systems

## Learning objectives

- Define eigenvalue and eigenvector geometrically (a direction the transformation only stretches).
- Find eigenvalues via the characteristic polynomial; find eigenvectors by solving (A − λI)x = 0.
- Diagonalize a matrix when possible; understand when it isn't.
- Use diagonalization to compute powers and exponentials of matrices.
- Analyze long-term behavior of discrete dynamical systems x_{k+1} = Ax_k.
- Handle complex eigenvalues geometrically (rotation + scaling).

## Prerequisites

- Chapters 1–8.

## Estimated time

10–12 hours. This is the densest chapter in the course.
