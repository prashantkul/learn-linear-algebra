# Chapter 7 — Orthogonality, Projections, Gram–Schmidt, QR, Least Squares

## Learning objectives

- Project a vector onto a subspace.
- Build an orthonormal basis from any basis using Gram–Schmidt.
- Factor a matrix as A = QR.
- Recognize an orthogonal matrix and its preserving properties (length, angle).
- Solve overdetermined systems via least squares; understand the normal equations.
- Define an inner product abstractly; treat function spaces with an L² inner product.

## Prerequisites

- Chapters 1–6.

## Estimated time

8–10 hours.
