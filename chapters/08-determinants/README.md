# Chapter 8 — Determinants: Algebra and Geometry

## Learning objectives

- Compute determinants of small matrices by cofactor expansion and by row reduction.
- State the multiplicative property: det(AB) = det(A)·det(B).
- Interpret det(A) geometrically as the signed volume scaling factor of A as a transformation.
- Use Cramer's Rule (and know when not to).

## Prerequisites

- Chapters 1–7.

## Estimated time

5–7 hours.
