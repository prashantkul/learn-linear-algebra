# Public NotebookLM Notebooks + Code Notebooks

One pre-built NotebookLM per chapter. Open the link to read the notes, listen to the audio overview, take the quiz, browse the mind map, view the slide deck and infographic, or chat with the sources. You can also click *"Make a copy"* in NotebookLM to clone it into your own account and customize.

Each chapter also has two Jupyter notebooks: a Python/NumPy one (numerical, with pictures) and a SageMath one (exact symbolic arithmetic). Open the Python notebook locally with `uv run jupyter lab`, or open the Sage notebook free in [CoCalc](https://cocalc.com/) (no install needed).

## What's in each NotebookLM

- The chapter's tutorial notes, worked examples, and exercises (markdown)
- Page extracts from the two anchor textbooks (Bretscher and Saveliev)
- A generated **audio overview** (deep-dive podcast — listen on your commute)
- A generated **quiz** to self-test
- A generated **mind map** of the chapter's concepts
- A generated **slide deck** for review
- A generated **infographic** for at-a-glance reference
- Chat-with-sources interface for "explain X", "give me another example of Y", etc.

## What's in each code notebook

- 🐍 **Python notebook** — NumPy and matplotlib. Numerical computation with pictures. Same flavor of code you'll write at work in any ML or scientific computing context.
- 🧮 **SageMath notebook** — exact symbolic arithmetic. Sage's first-class `Matrix`, `vector`, `var`, `QQ`, `QuadraticField`, etc. Proves things the Python notebook can only sample. Each chapter's Sage notebook also has a built-in exercise set with solutions.
